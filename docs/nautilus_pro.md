# 🧭 Modo Navegador Pro (Gestor de Archivos)

Este documento detalla la configuración avanzada de **Archivos (Nautilus)** en Zorin OS para maximizar la productividad mediante un flujo de trabajo de pestañas y jerarquía similar al de un navegador web.

## 🗂️ Gestión de Pestañas (Tabs)

El uso de pestañas permite gestionar múltiples directorios dentro de una sola ventana, evitando el desorden en el escritorio.

### Atajos Esenciales
| Acción | Atajo |
| :--- | :--- |
| **Nueva Pestaña** | `Ctrl + T` |
| **Cerrar Pestaña** | `Ctrl + W` |
| **Cambiar Pestaña Siguiente** | `Ctrl + PagAv (Page Down)` |
| **Cambiar Pestaña Anterior** | `Ctrl + PagRe (Page Up)` |
| **Reabrir Pestaña Cerrada** | `Ctrl + Shift + T` |

### Trucos de Ratón
*   **Apertura Instantánea:** Haz clic con el **botón central del ratón (la rueda)** sobre cualquier carpeta para abrirla inmediatamente en una pestaña nueva en segundo plano.
*   **Arrastrar y Soltar:** Puedes arrastrar archivos directamente a la cabecera de una pestaña para moverlos a ese directorio.

## 🌳 Navegación Jerárquica (Vista de Árbol)

Hemos activado la **Vista de Árbol** en el modo de lista. Esta funcionalidad es la que permite expandir carpetas sin entrar en ellas, ideal para mover archivos entre niveles profundos.

1.  Asegúrate de estar en el modo de **Vista de Lista** (el icono de las tres líneas horizontales arriba a la derecha).
2.  Verás una pequeña flecha `>` junto a los nombres de las carpetas.
3.  Al hacer clic, la carpeta se expandirá "hacia abajo", mostrando su contenido sin sacarte de tu ubicación actual.

## ⚡ Atajos de Navegación Pro

*   **Alt + Flecha Arriba:** Sube un nivel en la jerarquía de carpetas.
*   **Alt + Flecha Izquierda/Derecha:** Retrocede o Avanza en tu historial de navegación (como en Chrome).
*   **F3 (Panel Extra):** Abre una vista de panel dividido (Split view), muy útil para comparar dos carpetas en la misma ventana sin usar pestañas.
*   **Ctrl + L:** Enfoca la barra de direcciones para escribir una ruta manualmente.
